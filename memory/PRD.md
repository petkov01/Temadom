# TemaDom - Маркетплейс за строителство и ремонти

## Дата на създаване: 26.02.2026
## Последна актуализация: 26.02.2026

## Оригинално задание
Маркетплейс приложение за продажба на готови клиентски проекти (leads) на фирми и майстори в строителния сектор.

### Бизнес модел
- Pay-per-lead: €25 за контакт
- Месечен абонамент: €100 за пълен достъп
- Проектите се виждат, но контактите са скрити до плащане

## Потребителски персони
1. **Клиент** - търси майстори/фирми за ремонт
2. **Фирма/Майстор** - търси клиенти, закупува контакти
3. **Админ** - управлява платформата

## Имплементирани функции (Iteration 3)

### Навигация
- [x] **"Публикувай проект"** - оранжев бутон в менюто
- [x] **"Намери проекти"** - линк за фирми
- [x] **"Калкулатор"** - директен достъп до калкулатора

### Калкулатор (28 категории)
- Боядисване, шпакловка, замазка
- Ел. инсталации, ВиК
- Бетон, арматура, кофраж
- Плочки, подови настилки, изкъртване
- Зидария, топлоизолация
- **Изкопни работи, основи, покриви**
- **Дограма, гипсокартон, външна мазилка**
- **Канализация, огради, настилки двор**
- **Сухо строителство, дърводелство, хидроизолация**
- **Отопление, климатизация, басейни**

### Калкулатор на началната страница
- [x] Хубава секция с описание на ползите
- [x] 28 категории услуги
- [x] Цени по 9 региона (София, Пловдив, Варна...)
- [x] 3 нива качество (Икономичен, Стандартен, Премиум)
- [x] Примерна калкулация (8,556€ за 80м² апартамент)

### Проекти
- [x] До 10 снимки на проект (стаи, бани, чертежи)
- [x] **Оценка на бюджета** - вграден калкулатор при публикуване
- [x] Снимки се показват на картите на проекти
- [x] Оценката се показва в детайлите на проекта

### Портфолио
- [x] Фирмите качват снимки ПРЕДИ/СЛЕД от проекти

### Плащания
- [x] Stripe интеграция (test key)
- [x] Единичен контакт €25
- [x] Месечен абонамент €100

### Рейтинг система
- [x] Клиентите оставят отзиви
- [x] Отзивите НЕ могат да се изтриват от фирми

## Технологии
- Frontend: React 19, Tailwind CSS, Radix UI, Lucide Icons
- Backend: FastAPI, Python
- Database: MongoDB
- Плащания: Stripe
