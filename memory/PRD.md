# TemaDom - Маркетплейс за строителни услуги

## Бизнес модел
Платформата е БЕЗПЛАТНА за всички потребители.

## Потребителски роли
- **Клиенти:** Публикуват проекти, оценяват фирми
- **Фирми (с Булстат 9 цифри):** Портфолио, проекти, чат, Telegram
- **Майстори (без фирма):** Същите функции, без Булстат

## Завършени функции
- [x] 3 типа регистрация (Клиент/Фирма/Майстор) + Булстат
- [x] Ценови калкулатор (актуализирани цени 2025-2026)
- [x] "Намери майстор" страница с филтри
- [x] Чат система (без филтриране, със снимки)
- [x] Telegram бот (известия за нови проекти)
- [x] 12 подробни SEO статии (ключови думи за Google)
- [x] 37 автоматични SEO статии (професии + региони)
- [x] Безплатна платформа (без споменаване за бъдещо плащане)
- [x] Изчистени фалшиви данни (0 фирми, 0 проекти)
- [x] GA4 интеграция (G-526267282)

## SEO статии (ключови думи)
1. Цялостен ремонт на апартамент 2026 - цени кв.м
2. Ремонт на баня - цени, хидроизолация, фаянс
3. Ремонт на покрив - цени, керемиди, ламарина
4. Топлоизолация - EPS, XPS, спестявания
5. Хидроизолация - баня, покрив, основи
6. Ел. инсталация - цени за точка
7. Бетон и кофраж - цени м³, м²
8. Как да изберем строителна фирма
9. Ремонт на кухня - цени, етапи
10. Строителство на къща - цена кв.м
11. Замазка и изравняване на под
12. Боядисване - цени, видове боя

## Технологии
- Frontend: React, TailwindCSS, Shadcn/UI
- Backend: FastAPI, MongoDB, JWT
- Интеграции: GA4, Telegram Bot
