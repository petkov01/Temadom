# TemaDom - Маркетплейс за строителни услуги

## Оригинална идея
Платформа за свързване на клиенти със строителни фирми и майстори в България.

## Бизнес модел
**ТЕКУЩ:** Платформата е БЕЗПЛАТНА за всички потребители за ограничен период.

## Потребителски роли
- **Клиенти:** Публикуват проекти, оценяват фирми
- **Фирми (с Булстат 9 цифри):** Портфолио, проекти, чат, Telegram известия
- **Майстори (без фирма):** Същите функции, без Булстат
- **Админ:** Аналитика

## Основни функции (всички завършени)
- [x] 3 типа регистрация (Клиент / Фирма с Булстат / Майстор)
- [x] JWT автентикация
- [x] Публикуване на проекти (със снимки)
- [x] Ценови калкулатор (28 области, 28+ категории, PDF оферта)
- [x] Профили с портфолио и ревюта
- [x] Чат (без филтриране, със снимки)
- [x] Telegram бот (известия за нови проекти)
- [x] "Намери майстор" страница с филтри (тип/професия/град)
- [x] Цени актуализирани по пазарно проучване 2025-2026

## Ценови актуализации (Февруари 2026)
- Покриви: 50 €/м² (труд стандарт)
- Бетон: 70 €/м³
- Кофраж: 32 €/м²
- Хидроизолация: 18 €/м²
- Топлоизолация: 20 €/м²
- Арматура: 0.8 €/кг
- Ел. инсталация: 40 €/точка
- Стара Загора множител: 1.00

## Технологии
- Frontend: React, TailwindCSS, Shadcn/UI
- Backend: FastAPI, Python, MongoDB, JWT
- Интеграции: Stripe (деактивиран), GA4, Telegram Bot
