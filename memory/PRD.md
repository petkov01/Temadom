# TemaDom - Маркетплейс за строителство и ремонти

## Дата на създаване: 26.02.2026

## Оригинално задание
Маркетплейс приложение за продажба на готови клиентски проекти (leads) на фирми и майстори в строителния сектор.

### Бизнес модел
- Pay-per-lead: €25 за контакт
- Месечен абонамент: €100 за пълен достъп
- Проектите се виждат, но контактите са скрити до плащане

### Категории
15 строителни/ремонтни категории (електричество, ВиК, боядисване, бетон, арматура, плочки, подове, покриви, дограма, ОВК, изолация, събаряне, зидария, дърводелство, общо строителство)

## Потребителски персони
1. **Клиент** - търси майстори/фирми за ремонт
2. **Фирма/Майстор** - търси клиенти, закупува контакти
3. **Админ** - управлява платформата

## Основни изисквания
- [x] Регистрация и вход (клиенти и фирми)
- [x] Публикуване на проекти от клиенти
- [x] Листване на проекти с филтри
- [x] Заключени контакти до плащане
- [x] Stripe плащания (единичен контакт или абонамент)
- [x] Рейтинг система (несменяеми отзиви)
- [x] Профили на фирми
- [x] **Портфолио** - снимки преди/след за фирми
- [x] **Интерактивен калкулатор** - базиран на български пазарни цени

## Имплементирани функции
1. **Начална страница** - hero, статистики, категории
2. **Проекти** - листване, филтри, детайли със заключени контакти
3. **Фирми** - листване, профили, отзиви
4. **Калкулатор** - 12 категории услуги, 3 нива качество, 9 региона
5. **Портфолио** - качване на преди/след снимки
6. **Табла** - за клиенти (проекти) и фирми (контакти, портфолио)
7. **Плащания** - Stripe интеграция

## Технологии
- Frontend: React 19, Tailwind CSS, Radix UI
- Backend: FastAPI, Python
- Database: MongoDB
- Плащания: Stripe
