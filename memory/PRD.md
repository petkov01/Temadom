# TemaDom - Маркетплейс за строителство и ремонти

## Дата на създаване: 26.02.2026
## Последна актуализация: 27.02.2026 (Iteration 5)

## Оригинално задание
Маркетплейс приложение за продажба на готови клиентски проекти (leads) на фирми и майстори в строителния сектор.

### Бизнес модел
- Pay-per-lead: €25 за контакт
- Месечен абонамент: €100 за пълен достъп
- Първите 3 контакта са безплатни за нови фирми
- Калкулаторът е безплатен за клиенти, 5 безплатни за фирми, после €10

## Потребителски роли
1. **Клиент** - търси майстори/фирми за ремонт, безлимитен калкулатор
2. **Фирма/Майстор** - търси клиенти, закупува контакти, 5 безплатни калкулации

## ИМПЛЕМЕНТИРАНИ ФУНКЦИИ

### Калкулатор (ГЛАВНА ФУНКЦИЯ)
- [x] 28 категории услуги с регионални цени за 28 области на България
- [x] 3 нива качество (Икономичен, Стандартен, Премиум)
- [x] Труд или Труд + материали
- [x] PDF генериране на оферта
- [x] Лимит за фирми: 5 безплатни, после €10 за 5 допълнителни
- [x] Абонатите имат неограничен достъп

### Лого
- [x] Използва се ТОЧНОТО лого от потребителя (PNG от HEIC конверсия)
- [x] Показва се в навигацията и футъра

### Проекти
- [x] Създаване на проекти с до 10 снимки
- [x] Оценка на бюджета от калкулатора
- [x] Галерия с lightbox
- [x] Филтриране по категория, град, търсене

### Контакти за фирми
- [x] 3 безплатни контакта за нови фирми
- [x] Единичен контакт: €25
- [x] Абонамент: €100/месец за пълен достъп
- [x] Stripe интеграция за плащания

### Чат система
- [x] Съобщения между клиенти и фирми
- [x] Автоматично филтриране на контактна информация (телефони, имейли, социални мрежи)
- [x] Филтрирането работи ако фирмата НЕ Е платила за достъп
- [x] Разговори с непрочетени брояч
- [x] Polling на всеки 5 секунди за нови съобщения

### Портфолио за фирми
- [x] Снимки ПРЕДИ/СЛЕД от проекти

### Страница "Услуги"
- [x] 13+ подробни описания на услуги
- [x] Филтриране по категории (Довършителни, Инсталации, Груб строеж, Изолации)
- [x] Стъпки на изпълнение, материали, съвети, предупреждения

### Навигация
- [x] "Публикувай проект" - оранжев бутон
- [x] "Намери проекти" - за фирми
- [x] "Калкулатор" - директен достъп
- [x] "Услуги" - справочник
- [x] "Съобщения" - чат (за логнати потребители)

## Технологии
- Frontend: React 19, Tailwind CSS, Shadcn/UI
- Backend: FastAPI, Python
- Database: MongoDB
- Плащания: Stripe
- PDF: fpdf2
