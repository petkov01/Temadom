# TemaDom - Маркетплейс за строителни услуги

## Оригинална идея
Платформа за продажба на клиентски проектни лийдове на строителни фирми и майстори в България.

## Бизнес модел
**ТЕКУЩ (Февруари 2026):** Платформата е БЕЗПЛАТНА за всички потребители за ограничен период.

## Потребителски роли
- **Клиенти:** Публикуват проекти, оценяват фирми
- **Фирми (с Булстат):** Разглеждат проекти, чат, портфолио - изискват ЕИК 9 цифри
- **Майстори (без фирма):** Същите функции като фирми, без Булстат
- **Админ:** Достъп до аналитика

## Основни функции
- [x] Регистрация с 3 типа (Клиент / Фирма с Булстат / Майстор)
- [x] JWT автентикация
- [x] Публикуване на проекти (със снимки)
- [x] Интерактивен ценови калкулатор (28 области, 28+ категории)
- [x] PDF генериране на оферта (с кирилица)
- [x] Профили на фирми/майстори с портфолио
- [x] Система за оценки/ревюта
- [x] Чат система (без филтриране, със снимки)
- [x] Регистрация по град/населено място
- [x] Telegram бот интеграция (известия за нови проекти)
- [x] Безплатна платформа (премахнати всички платежни бариери)
- [x] Булстат валидация (9 цифри, задължителен за фирми)

## Технологии
- **Frontend:** React, TailwindCSS, Shadcn/UI, lucide-react
- **Backend:** FastAPI, Python, MongoDB, JWT
- **PDF:** fpdf2 с Cyrillic поддръжка
- **Интеграции:** Stripe (деактивиран), GA4 (G-526267282), Telegram Bot

## Архитектура
```
/app/backend/server.py - Всички backend routes
/app/frontend/src/App.js - Основен файл с routing и компоненти
/app/frontend/src/components/ - Калкулатор, Чат и др.
/app/frontend/src/pages/ - SEO страници
/app/frontend/src/data/ - Блог и региони данни
```
